The HtmlDocument received from the web get request. Will be null until the DoGet method is called.
HasNextPage
Returns false if the element for the XPath NEXT_BUTTON has a 'disable' class attribute, the HtmlDocument is null, or if the XPath returns a null node. Otherwise returns true.
Items
The dictionary containing the items parsed from the HtmlDocument by the parse provider. Will be null until the DoGet method is called.
